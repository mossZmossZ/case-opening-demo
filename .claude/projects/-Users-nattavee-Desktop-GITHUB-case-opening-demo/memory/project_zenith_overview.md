Building a case-opening / lucky-draw web application for corporate events. Two UIs: User (spin reel to win prizes) and Admin (manage prizes, rates, inventory). Design follows "The Kinetic Foundry" industrial aesthetic from stitch templates. Tech: React, Node.js, MongoDB, Redis (Phase 3+), Kubernetes (Phase 4). Four phases scaling from local npm to full K8s with Istio observability.

**Why:** Corporate event demo for Nutanix Cloud Native & AI Innovation Day. Needs to scale from quick prototype to production-grade showcase.

**How to apply:** Follow the 4-phase roadmap. Only load the phase doc currently being worked on (lazy-load pattern to save tokens).
